## Description
A terminal-based Expense Tracker application developed in Python. This project allows users to record daily expenses, calculate a running total, and display a complete expense summary.

The system uses an interactive input process with validation to prevent invalid entries and improve user experience.

---

## Features

- Add expense amounts
- Auto-generated Expense IDs
- Running total calculation
- Input validation
- Error handling using try-except
- View complete expense summary
- Interactive terminal interface

---

- Lists

## Program Workflow

1. User enters expense amount  
2. System validates input  
3. Expense gets stored  
4. Running total updates automatically  
5. User types "done" to finish  
6. Final expense summary appears
