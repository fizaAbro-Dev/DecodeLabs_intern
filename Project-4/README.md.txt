## Description

A terminal-based General Knowledge Quiz application developed in Python. The game asks users multiple-choice style questions, checks answers, calculates scores, and provides performance feedback at the end.

The project demonstrates user interaction, input handling, conditional logic, and score tracking.

---

## Features

- Interactive quiz system
- Multiple general knowledge questions
- Input validation using string processing
- Score tracking system
- Instant feedback after each question
- Performance message based on score
- User-friendly terminal interface

---

## Technologies Used

- Python
- Variables
- Conditional Statements
- User Input Handling
- String Methods
- Basic Score System

## Program Workflow

1. Display welcome message  
2. Ask quiz questions  
3. Receive user answers  
4. Process and compare responses  
5. Update score  
6. Display final score and feedback
